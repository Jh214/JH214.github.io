description: NOVER 프로젝트는 저의 첫 번째 외주 개발 프로젝트로, 실제 운영되는 의류 기반 쇼핑몰 플랫폼의 백엔드 전반을 담당하여 개발한 프로젝트입니다.<br>전체 기능은 REST API 기반으로 구성되어 있으며, 회원·상품·장바구니·주문·결제·배송·프로모션 등 쇼핑몰 핵심 도메인을 모두 포함하고 있습니다.<br><br>로그인은 JWT 기반으로 구현했으며, 액세스 토큰은 15분 유효 기간을 갖습니다.<br>리프레시 토큰은 HttpOnly 쿠키에 저장되어 자동 재발급이 가능하도록 설계했습니다.<br>회원가입 시 별도의 인증 절차는 존재하지 않으며, 아이디는 이메일을 그대로 사용합니다.<br>비밀번호 찾기는 사용자의 이메일로 임시 비밀번호를 발급해 전송하는 방식으로 구현했습니다.<br>개인정보 수정 및 비밀번호 변경 시에는 비밀번호 재확인을 거쳐 접근하도록 하여 보안을 강화했습니다.<br><br>장바구니는 옵션별 수량 관리, 재고 검증, 비로그인/로그인 병합 정책을 포함해 구현했습니다.<br>배송비는 기본 배송비, 무료 배송 기준, 도서산간 추가 비용 등 정책 기반으로 계산됩니다.<br>쿠폰은 상품·카테고리 조건, 최소 금액 조건, 중복 적용 여부 등 다양한 정책을 반영해 개발했습니다.<br><br>결제는 PortOne V2와 연동하였으며, PG 결제 금액과 주문 금액의 일치 여부 검증, 결제 성공·실패·취소 상태 처리 등 안정적인 결제 플로우를 구성했습니다.<br><br>관리자 페이지에서는 메인 노출 상품 관리, 배너 관리, 프로모션 정책 설정, 메인 헤더 텍스트 관리, 방문자 카운팅(UV), 대시보드 통계 API 등을 제공합니다.<br>게시판(공지·이벤트·문의)과 리뷰 기능도 구현하여 운영자가 다양한 콘텐츠를 효율적으로 관리할 수 있도록 구성했습니다.<br><br>이 프로젝트를 통해 도메인 설계, 인증·인가 구조 설계, 예외 처리, API 문서화, 폴더 구조 구성, 서버 배포 및 운영, 그리고 팀원 간 소통의 중요성을 크게 체감했습니다.<br><br>클라이언트측 요청으로 운영서버는 운영 중단하였고 개발서버만 남아있는 상태입니다.
